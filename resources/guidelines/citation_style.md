# Citation Style Guidelines (SRH Fernhochschule – APA / DGPs, In-Text)

> This file defines the **only** allowed citation style in this repository:  
> APA / DGPs-style **author–year in-text citations**, based on the SRH Studienbrief *Wissenschaftliches Arbeiten und Schreiben* (Kapitel 3 & 3.4.1).

---

## 1. Fundamental Principles

1. **Every borrowed idea or wording must be cited.**
2. Citations are given **in the text**, not in footnotes.
3. The **reference list** contains the full bibliographic data; the text only contains short author–year–page hints.
4. Citations must be **traceable**: the reader must be able to find the exact passage in the original.

---

## 2. In-Text Citation Format

### 2.1 Basic pattern

```text
(Autor, Jahr, S. xx)
(Autor1 & Autor2, Jahr, S. xx–yy)
(Autor1, Autor2 & Autor3, Jahr, S. xx)
(Organisation, Jahr, S. xx)
```

### Examples

**Paraphrase:**
... wie in neueren Studien gezeigt wurde (Müller, 2020, S. 15).

**Direct quote:**
„Zitattext ...“ (Schmidt, 2019, S. 44).

### 2.2 Authors

**One author:**
(Müller, 2020, S. 15)

**Two authors:**
(Müller & Schmidt, 2021, S. 22)

**Three or more authors:**
First citation in a section can be written with all authors if desired.
Otherwise use: (Müller et al., 2022, S. 5)

### 2.3 Multiple works

**Same author, different years:**
(Müller, 2018, 2019)

**Different authors:**
(Müller, 2020; Schmidt, 2021)

Sort by year or alphabetically; stay consistent.

### 2.4 Page numbers

- Direct quotes: page number mandatory.
- Close paraphrases of specific passages: page number strongly recommended.
- General references to a whole work: page number may be omitted.

**Format:**
- S. 5
- S. 5–6
- S. 101–103

---

## 3. Reference List (Literaturverzeichnis)

### 3.1 General rules

- Start on a new page, titled *Literaturverzeichnis* (or *References* if English).
- Alphabetical order by the first author's surname.
- Works by the same author are sorted chronologically (oldest first).
- Only include works that are actually cited in the text.

### 3.2 Basic patterns (German APA)

**Book (monograph)**
```
Nachname, Vorname-Initiale. (Jahr). Titel des Buches: Untertitel. Auflage (falls nicht 1.). Verlag.
```

**Edited volume**
```
Nachname, Vorname-Initiale (Hrsg.). (Jahr). Titel des Sammelbandes. Verlag.
```

**Chapter in edited volume**
```
Nachname, Vorname-Initiale. (Jahr). Titel des Beitrags. In Vorname-Initiale Nachname (Hrsg.), Titel des Sammelbandes (S. xx–yy). Verlag.
```

**Journal article**
```
Nachname, Vorname-Initiale. (Jahr). Titel des Artikels. Zeitschriftentitel, Bandnummer(Heftnummer), xx–yy.
```

**Online source** (only if not simply the PDF of a print work)
```
Nachname, Vorname-Initiale. (Jahr). Titel des Dokuments. Website/Organisation. URL
```
- If available, use DOI instead of a bare URL.
